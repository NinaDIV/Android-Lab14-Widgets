# Android Lab 14 - App Widgets

Laboratorio Android sobre desarrollo de Widgets de pantalla de inicio. Implementacion de App Widgets con actualizacion dinamica y configuracion personalizada por el usuario.

## Temas cubiertos
- AppWidgetProvider y ciclo de vida de Widgets
- RemoteViews para actualizar la UI del Widget
- Configuracion del Widget con AppWidgetProviderInfo
- Actualizacion periodica del Widget
- Widget configurable por el usuario
